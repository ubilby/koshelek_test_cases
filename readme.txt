1. Решение первого задания - файл test1.py. запускается командой pytest test1.py -v.
Зависимости прикладываю -requirements.txt (Selenium - для второго теста). Писал на python3.8.3
Тест представляет из себя восемь асинхронно запущенных тестов, которые асинхронно обращаются
к coinmarketcapAPI. тест считается успешным, если каждый из запусков уложился в 500мс,
10кб, умеет данные по десяти тикетам с топ-объемом за 24часа, если 80%перцентиль всех ответов
ниже 450мс и скорость ответа сервера выше пяти в секунду - собственно, собственно если какое-то
не выполнится сообщение при Failed сообщит какое. Справился с подключением к API и проверкой
условий первого задания за 1,5-2 часа, часов 10-12 тщетно пытался запустить этот тест, асинхронно и
используя тестовые библиотеки, но оставил эти попытки и за еще два часа привел его к существующему
виду.

2. Решение второго задания - файл test2.py. запускается командой pytest test2.py -v.
Из зависимостей - только тест и установленный geckodriver для Firefox. Писал на python3.8.3
Тест представляет из себя подключение к сайту, нахождению кнопки выбора языка,
цикл проходящий по всем языкам и сверяющий содержимое заголовка сайта с выбранным языком.
На задание потратил часа 3-4. Была мысль прогнать по двум циклам, чтобы из каждый язык
был выбран из всех языков, но ожидание выполнение теста заняло бы минут 25.
